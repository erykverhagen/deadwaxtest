# Fresh repo deploy steps

1. Clear the GitHub Pages repo.
2. Upload everything from this ZIP.
3. Confirm `supabase-config.js` contains the public project URL and publishable key.
4. Open the GitHub Pages URL.
5. Sign in.
6. Confirm records load from Supabase.
7. Edit one harmless record and refresh to confirm persistence.

The `/tools/importer.html` file is parked for migration only. It is not part of the regular app flow.
